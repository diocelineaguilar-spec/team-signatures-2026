Email Signature Generator

Savas Labs — generate your team signature in seconds

How to install

  1. Fill out the form below and click Generate Signature
  2. Click Copy Signature in the preview — or download the HTML file and open it in Chrome
  3. Select all (⌘A) and copy (⌘C) the page content
  4. Go to Gmail Settings → General and paste into the Signature field
  5. Check "Insert this signature before quoted text in replies"
  6. Scroll down and click Save Changes
